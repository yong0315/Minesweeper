# Minesweeper
조작 방법

== 시작 화면 ==
[1] 쉬움 (9x9, 지뢰 10개)
[2] 보통 (16x16, 지뢰 40개)
[3] 어려움 (30x16, 지뢰 99개)
[4] 커스텀 (가로 9~60 x 세로 9~60, 지뢰 1 ~ 3599개)

== 커스텀 설정 화면 ==
[space] 설정 변환 (가로 ==> 세로, 세로 ==> 지뢰 개수) / 설정 완료 후 게임 시작
[backspace] 숫자 한자리씩 지우기
[esc] 시작 화면

== 게임 화면 ==
[1] 클릭 모드
[2] 깃발 모드
[방향키] 커서 이동
[space] 닫힌 칸 열기 / 깃발 설치 (이미 설치되어 있는 경우 깃발 제거)
[esc] 시작 화면
